Welcome to Our Application

This Flask application implements a prototype for a Firewall network on a application with three segments:

  • The external network
  • The DMZ (Demilitarized Zone)
  • The internal network
  • The application aims to demonstrate security features such as Content Security Policy (CSP), HTTP Strict Transport Security (HSTS), Clickjacking protection, error handling, and logging
